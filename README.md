## Project Overview
**AgriChatBot** is a Python web application designed to empower farmers by providing quick and reliable agricultural information. It leverages a **Retrieval-Augmented Generation (RAG) model** to answer farmers' queries. The model retrieves relevant information from a collection of agricultural websites and PDF documents, then generates precise and comprehensive answers tailored to each user's question.

This project is a personalized version of AgriGenius, enhanced for better usability and clarity.

---

## Features

- Fetch content from specified agricultural websites.
- Extract and process text from PDF files.
- Initialize a vector store for efficient information retrieval.
- Set up a Retrieval QA chain using a language model to answer queries related to agriculture.
- Interactive web interface to chat with the system.

Follow these steps to set up the project locally:

```bash

git clone https://github.com/ZSaba114/AgriChatBot.git

Configuration
The language model used: meta-llama/Llama-2-70b-chat-hf

The application uses the Together API for LLM services.

Add your own Together API key in chat2.py:

llm = Together(
    model="meta-llama/Llama-2-70b-chat-hf",
    max_tokens=512,
    temperature=0.1,
    top_k=1,
    together_api_key="YOUR_Together_API_KEY"
)
Ensure requirements.txt includes all necessary packages: Flask, requests, PyPDF2, langchain, chromadb, etc.
